# CuspControl
Local and Real-Time applications for controlling ASACUSA's Cusp Experiment
### Requirements
LabVIEW 2018 SP1 or greater\
LabVIEW Real-Time Module 2018 SP1 or greater\
AMC library (install using VI Package Manager)\
RT target (we use PXIe 8821)\
At least one macro (make one using MacroEditor.vi)
